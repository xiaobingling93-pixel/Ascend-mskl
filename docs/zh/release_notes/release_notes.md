## 特性变更说明

### 26.0.0

无新增特性。

### 8.3.0

首次发布，功能点如下：

1. 提供tiling_func和get_kernel_from_binary接口，支持调用msOpGen工程中的tiling函数以及用户自定义的Kernel函数，便于快速调测。

2. 提供了autotune系列接口，支持对模板库算子进行代码替换、编译、运行以及性能对比，便于高效调优。
